# Governance (Source of Truth + Enforcement)

## Source of truth
- Behavior rules: `docs/AGENT.md`
- Team SOP: `docs/AGENTS.md`
- Stage order: `docs/RUNBOOK.md`
- Proof gates: `docs/VERIFY.md`
- Prompts: `docs/PROMPTS.md`
- Learnings: `docs/LEARNED.md`
- Repo reality map (mandatory): `docs/REPO_MAP.md`

## Enforcement
- Stage 00 is mandatory before any edits.
- No stage is DONE without proof + QA signoff + Stage Report.

## Conflict resolution
- `docs/AGENT.md` wins if anything conflicts.
- If a path/config is uncertain: confirm via repo search output (Repo‑Truth).

## Blocker rule
If blocked, output a BLOCKER REPORT (template in `docs/PROMPTS.md`) and stop.
